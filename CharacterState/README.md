# 캐릭터 상태 관리

게임 캐릭터의 상태를 관리하는 프로그램을 작성하세요.

캐릭터는 다음 상태 중 하나를 가질 수 있습니다:

- Idle (대기) - 값: 0
- Walking (걷기) - 값: 1
- Running (달리기) - 값: 2
- Jumping (점프) - 값: 3
- Attacking (공격) - 값: 4
- Dead (사망) - 값: 5

프로그램은 다음 기능을 수행해야 합니다:

1. 캐릭터의 현재 상태를 출력하는 기능
2. 사용자가 선택한 상태로 변경하는 기능
3. 모든 상태 목록과 해당 정수 값을 출력하는 기능
4. 상태에 따른 행동 설명을 출력하는 기능

프로그램 흐름:

1. 캐릭터는 Idle 상태로 시작
2. 메뉴를 표시하고 사용자 입력을 받음
3. 선택에 따라 적절한 동작을 수행
4. 종료를 선택할 때까지 반복

## 예상 실행 결과

```
=== 캐릭터 상태 관리 ===
현재 상태: Idle

1. 상태 변경
2. 상태 목록 보기
3. 현재 행동 보기
4. 종료
선택: 2

=== 상태 목록 ===
Idle = 0
Walking = 1
Running = 2
Jumping = 3
Attacking = 4
Dead = 5

현재 상태: Idle

1. 상태 변경
2. 상태 목록 보기
3. 현재 행동 보기
4. 종료
선택: 1

변경할 상태 번호 입력 (0-5): 4
상태가 Attacking(으)로 변경되었습니다.

현재 상태: Attacking

1. 상태 변경
2. 상태 목록 보기
3. 현재 행동 보기
4. 종료
선택: 3

[행동] 적을 공격합니다!

현재 상태: Attacking

1. 상태 변경
2. 상태 목록 보기
3. 현재 행동 보기
4. 종료
선택: 1

변경할 상태 번호 입력 (0-5): 5
상태가 Dead(으)로 변경되었습니다.

현재 상태: Dead

1. 상태 변경
2. 상태 목록 보기
3. 현재 행동 보기
4. 종료
선택: 3

[행동] 사망 상태입니다. 행동할 수 없습니다.

현재 상태: Dead

1. 상태 변경
2. 상태 목록 보기
3. 현재 행동 보기
4. 종료
선택: 4

프로그램을 종료합니다.
```
