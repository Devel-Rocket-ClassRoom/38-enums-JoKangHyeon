# 열거형 형식 사용하기

## 사용 방법

1. Visual Studio에서 `Chapter38-Enums.slnx`를 엽니다
2. 솔루션 탐색기에서 프로젝트를 선택하고 `README.md`를 읽습니다
3. `Program.cs`에 코드를 작성합니다
4. 해당 프로젝트를 시작 프로젝트로 설정하고 실행합니다 (`Ctrl+F5`)

## 코딩 연습

`CodingPractice` 프로젝트의 README를 따라 교안 예제 코드를 직접 작성하고 실행합니다.

| 프로젝트 | 내용 |
|----------|------|
| CodingPractice-01 | 교안 코드 따라치기 - 열거형 기초, 값 지정, Enum 클래스, 변환 (10개 과제) |
| CodingPractice-02 | 교안 코드 따라치기 - 열거형 기본 값, Enum.GetValues (2개 과제) |
| CodingPractice-03 | 교안 코드 따라치기 - 값 지정, 플래그 열거형 (5개 과제) |
| CodingPractice-04 | 교안 코드 따라치기 - switch 문, 게임 활용 (4개 과제) |

## 연습문제

각 프로젝트의 README에 있는 문제를 읽고 코드를 작성합니다.

| 프로젝트 | 문제 |
|----------|------|
| CharacterState | 캐릭터 상태 관리 |
| FilePermission | 파일 권한 관리 |
